# [Chat] Template Text Folders

完整源码请查看 [`dev` 分支](https://github.com/0-V-linuxdo/Chat_Template_Text_Folders/tree/dev)。

<details open>
<summary>简介</summary>
  
- 在AI网页底部，添加 `提示词模版` 文件夹+按钮，
- 减少重复输入，提升输入效率！
