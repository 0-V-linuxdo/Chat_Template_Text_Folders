<details open>
<summary>简介</summary>
  
- 在AI网页底部，添加 `提示词模版` 文件夹+按钮，
- 减少重复输入，提升输入效率！

<summary>支持的浏览器</summary>
  
- Chrome
- Firefox
- Safari

</details>

<details>
